# Admissions Lead Conversion Analysis

## Overview
This project analyzes large-scale admissions datasets to identify enrollment trends, conversion gaps, and applicant behavior patterns using exploratory and comparative analytics techniques.

The analysis focuses on understanding lead conversion performance, regional trends, and operational insights that can support data-driven admission strategies.

---

## Objectives
- Analyze admission lead conversion trends
- Identify enrollment and applicant behavior patterns
- Perform exploratory data analysis on large-scale datasets
- Generate actionable insights for operational decision-making

---

## Dataset Information
- Dataset Size: 1M+ admission records
- Data Type: Admissions and lead conversion data
- Tools Used: Python, Excel

---

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Excel

---

## Methodology
1. Data Cleaning & Preprocessing
2. Exploratory Data Analysis
3. Conversion Trend Analysis
4. Regional & Behavioral Analysis
5. Data Visualization & Insight Generation

---

## Key Insights
- Identified conversion gaps across admission stages
- Analyzed applicant behavior and enrollment trends
- Generated visual insights to support admission decision-making
- Performed large-scale data handling and preprocessing workflows
